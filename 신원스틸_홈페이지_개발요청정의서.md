## 2. 레이아웃 구조 (SK하이닉스 참조)

SK하이닉스(skhynix.com)의 레이아웃 패턴을 그대로 따른다.

```
[상단 유틸리티 바]         ← 작은 글씨로 연락처 / 사이트맵 링크
[GNB — 로고 + 풀메뉴]     ← 스크롤 시 상단 고정(sticky), 메가메뉴 드롭다운
[히어로 섹션]              ← 전체 화면 높이, 배경 이미지 또는 영상 플레이스홀더
[섹션 1 — 회사 소개]       ← 좌: 텍스트 + 링크 / 우: 이미지 (SK하이닉스 DBL 방식)
[섹션 2 — 경영이념]        ← 풀 와이드 컬러 배경 + 텍스트 중앙 정렬
[섹션 3 — 취급강종]        ← 가로 스크롤 카드 or 3열 그리드
[섹션 4 — 생산설비]        ← 좌: 이미지 / 우: 텍스트 + 리스트 (섹션1 미러)
[섹션 5 — 뉴스 / 공지]     ← 카드 3열 (추후 확장용, 현재는 언론 기사 3건)
[섹션 6 — 위치 / CTA]      ← 지도 영역 + 견적의뢰 버튼
[푸터]                     ← 로고 + 링크 + 주소 + 카피라이트
```

## 3. 디자인 가이드 (토스 스타일)

### 색상
```
Primary Blue  : #1B64DA   ← 메인 포인트 컬러
Dark Text     : #191F28   ← 제목, 본문 주요 텍스트

Border        : #E5E8EB   ← 구분선, 테두리
White         : #FFFFFF   ← 기본 배경
```

### 타이포그래피
```
폰트: Pretendard (CDN: https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/static/pretendard.css)
제목(H1)  : 52px / font-weight 700 / line-height 1.2
제목(H2)  : 36px / font-weight 700 / line-height 1.3
섹션타이틀: 28px / font-weight 600
본문      : 16px / font-weight 400 / line-height 1.7
소제목    : 13px / font-weight 500 / letter-spacing 2px / 대문자 / color: #1B64DA
```

### 디자인 원칙
- 여백을 넉넉하게 (섹션 상하 padding: 100px 이상)
- 그림자 최소화 — box-shadow 사용 시 `0 2px 8px rgba(0,0,0,0.06)` 수준
- 버튼: 라운드 없음(border-radius: 6px) 또는 pill형(border-radius: 100px) 두 가지
- 애니메이션: 스크롤 진입 시 fade-up (transform + opacity, 0.5s ease)
- 이미지 영역: 실제 이미지 없을 경우 회색 placeholder (#E5E8EB 배경)
